​						`touch a1.c`

​						将文件提交到暂存区
​						`git add a1.c`

​						提交操作
​						`git commit -m"提交描述"`

​						关联远程仓库

​						`git remote add origin git@github.com:Fixer/esp.git`

​						本地库推送到远程库

​						`	git rm a1.php`
